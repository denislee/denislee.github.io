Com a minha ideia de reservar um computador para usar exclusivamente para realizar as transmissões das minhas lives na Twitch, resolvi instalar um Manjaro e depois fazer a atualização do seu WM do i3 para o Sway. Seguem as instruções.

Maioria das instruções estão aqui: https://wiki.archlinux.org/index.php/Sway

Outra boa referência é o endereço: https://bennetthardwick.com/blog/change-to-sway-arch-linux/

instalar o pacote sway yay -S sway

instalar os outros pacotes:

yay -S wlroot-git swaylock swayidle
alcritty waybar xorg-xwayland
azote swaybg wlogout
fuzzel
qt5-wayland qt6-wayland

Adicionar o código para abrir automaticamente o Sway assim que ligar o computador. adicionar no /etc/profile

if [ -z $DISPLAY ] && [ "$(tty)" == "/dev/tty1" ]; then exec sway fi

Remover o lightdm e o lightdm-gtk-greeter com o comando:

yay -R lightdm lightdm-settings lightdm-slick-greeter

Gosto de inverter o SUPER do ALT, então no .config/sway/config

xkb_options altwin:swap_alt_win
