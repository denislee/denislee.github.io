https://github.com/swaywm/sway/wiki

Rotacionar a tela no TTY

Adicione no arquivo /etc/default/grub

GUB_CMDLINE_LINUX_DEFAULT="fbcon=rotate:1"

Rotacionar a tela no Sway

No terminal

swaymsg output LVDS-1 transform 180

Ou no .config/sway/config

output LVDS-1 transform 90

Para pegar os outputs

swaymsg -t get_outputs

Trocar o kp_enter por return

Criar o arquivo .xkb/symbols/minibook-keyfix

partial modifier_keys
xkb_symbols "basic" {
  include "us"
  key <KPEN> { [ Return ] };
};

Adicionar no .config/sway/config

input * {
  xkb_layout minibook-keyfix
}

Rotacionar o touch

Pegar o input com swaymsg -t get_inputs e o output swaymsg -t get_outputs

swaymsg input 1386:230:Wacom_ISDv4_E6_Pen map_to_output eDP-1
