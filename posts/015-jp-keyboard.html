Wayland JP keyboard

Com a vantagem de usar as teclas adicionais do teclado japones como os modificadores do sway (pois eu não escrevo japonês)

No arquivo .config/sway/config adicione:

input * {
  xkb_layout jp-mod
  xkb_options caps:ctrl_modifier
}

Veja que ele faz referencia ao jp-mod

O arquivo fica em .xkb/symbols/jp-mod com o conteúdo:

partial modifier_keys
xkb_symbols "basic" {
  include "us(intl)"
  key <MUHE> { [ Super_L ] };
  key <HENK> { [ Super_R ] };
  modifier_map Mod4 { <MUHE>, <HENK> };
};
