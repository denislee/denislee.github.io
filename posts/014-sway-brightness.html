Brightness

Como configurar as teclas de atalho do laptop para controlar o brilho no Sway.

Adicione o bind das teclas de atalho para o comando do light

bindsym XF86MonBrightnessUp exec "sudo light -A 10"
bindsym XF86MonBrightnessDown exec "sudo light -U 10"

Para descobrir o valor do brilho atual:

cat /sys/class/backlight/intel_backlight/brightness

Para descobrir o valor do brilho máximo:

cat /sys/class/backlight/intel_backlight/max_brightness

Permissão para uso do light

Crie um arquivo /etc/sudoers.d/20-light com o conteúdo:

%wheel ALL=(root) NOPASSWD: /usr/bin/light

Sound

Instale o pulse audio:

yay -S pavucontrol pulseaudio pulseaudio-alsa

Inicie o serviço:

pulseaudio --start

Adicione as teclas de atalho:

bindsym XF86AudioLowerVolume exec "amixer -D pulse sset Master 5%-"
bindsym XF86AudioRaiseVolume exec "amixer -D pulse sset Master 5%+"
