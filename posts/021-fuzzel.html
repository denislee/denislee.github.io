Fuzzel

Instalação do Fuzzel

https://codeberg.org/dnkl/fuzzel

Criação do diretório

mkdir -p bld/release && cd bld/release

Realização da configuração

meson --buildtype=release \
    -Denable-cairo=disabled \
    -Denable-png=disabled \
    -Denable-svg=disabled \
    --prefix=/usr \
    ../..

Fazer o build

ninja

Só copiar para o /usr/local/bin

sudo cp ./fuzzel /usr/local/bin/
